Introduction

Python is a high-level, interpreted programming language created by Guido van Rossum and first released in 1991.

Python's design philosophy emphasizes code readability with its notable use of significant whitespace. Its language constructs and object-oriented approach aim to help programmers write clear, logical code for small and large-scale projects.

Python is dynamically typed and garbage-collected. It supports multiple programming paradigms, including:

  • Procedural
  • Object-oriented
  • Functional programming

Python is often described as a "batteries included" language due to its comprehensive standard library.

Getting Started

Installing Python is the first step to begin programming. Visit python.org to download the latest version for your operating system.

Once installed, you can verify the installation by opening a terminal or command prompt and typing:

python --version

To write your first program, create a file named hello.py with the following content:

print("Hello, World!")

Run the program from the terminal with the command:

python hello.py
Data Types

Python has several built-in data types:

  • Numeric Types: int, float, complex
  • Text Type: str
  • Sequence Types: list, tuple, range
  • Mapping Type: dict
  • Set Types: set, frozenset
  • Boolean Type: bool
  • Binary Types: bytes, bytearray, memoryview

Here are some examples of variable assignments:

x = 5 # int y = 3.14 # float name = "Python" # str is_cool = True # bool

Lists are ordered and changeable collections:

fruits = ["apple", "banana", "cherry"] fruits.append("orange") print(fruits[1]) # Output: banana

Dictionaries store key-value pairs:

person = { "name": "John", "age": 30, "city": "New York" } print(person["name"]) # Output: John
Control Flow

Control flow statements determine the order in which your code executes. Python has several control flow statements:

Conditional statements with if, elif, and else:

x = 10 if x > 10: print("x is greater than 10") elif x == 10: print("x is equal to 10") else: print("x is less than 10")

For loops for iteration:

for i in range(5): print(i) # Outputs 0, 1, 2, 3, 4

While loops continue until a condition is false:

count = 0 while count < 5: print(count) count += 1

Use the break statement to exit a loop prematurely:

The continue statement skips the current iteration and moves to the next one.

Functions

Functions are blocks of reusable code that perform a specific task. They help organize your code and make it more modular.

To define a function in Python, use the def keyword:

def greet(name): return f"Hello, {name}!" # Call the function message = greet("Alice") print(message) # Output: Hello, Alice!

Functions can have default parameter values:

def greet(name, greeting="Hello"): return f"{greeting}, {name}!" print(greet("Bob")) # Output: Hello, Bob! print(greet("Bob", "Hi")) # Output: Hi, Bob!

Python functions can return multiple values using tuples:

def get_dimensions(): return 100, 50 width, height = get_dimensions() print(width) # Output: 100 print(height) # Output: 50
Object Oriented Programming

Python is an object-oriented programming language. Everything in Python is an object, including strings, lists, dictionaries, and functions.

You can define your own classes to create custom objects:

class Person: def __init__(self, name, age): self.name = name self.age = age def greet(self): return f"Hello, my name is {self.name}" # Create an instance of the Person class alice = Person("Alice", 30) print(alice.greet()) # Output: Hello, my name is Alice

Inheritance allows you to create a new class that inherits attributes and methods from an existing class:

class Student(Person): def __init__(self, name, age, grade): super().__init__(name, age) self.grade = grade def study(self): return f"{self.name} is studying hard!" bob = Student("Bob", 15, "10th") print(bob.greet()) # Inherited from Person class print(bob.study()) # Method from Student class
Modules and Packages

Modules are Python files containing functions, classes, and variables that you can import and use in your programs.

To import a module, use the import statement:

import math print(math.sqrt(16)) # Output: 4.0

You can import specific functions or classes from a module:

from math import sqrt, pi print(sqrt(16)) # Output: 4.0 print(pi) # Output: 3.141592653589793

Packages are directories containing multiple module files and a special __init__.py file. They help organize related modules.

The Python Standard Library includes many useful modules and packages for common tasks:

  • os and sys for operating system interfaces
  • datetime for date and time handling
  • json for JSON data processing
  • re for regular expressions
  • random for generating random numbers
Exception Handling

Exception handling allows you to gracefully handle errors that occur during program execution.

Use try, except, else, and finally blocks to handle exceptions:

try: x = 10 / 0 # This will raise a ZeroDivisionError except ZeroDivisionError: print("Cannot divide by zero!") else: print("Division successful") # This will run if no exception occurs finally: print("This always executes") # This will always run

You can catch multiple exception types:

try: num = int(input("Enter a number: ")) result = 10 / num except ValueError: print("That's not a valid number!") except ZeroDivisionError: print("Cannot divide by zero!")

To raise an exception manually, use the raise statement:

def validate_age(age): if age < 0: raise ValueError("Age cannot be negative") return age

Custom exception classes can be created by inheriting from the Exception class:

class CustomError(Exception): def __init__(self, message): self.message = message super().__init__(self.message)
